# Quick Installation Guide

## Step-by-Step Installation

### 1. Prepare the Extension
- Make sure all files are in the `WP AUTOMATE` folder
- Verify you have these files:
  - manifest.json
  - popup.html
  - popup.css
  - popup.js
  - content.js
  - background.js
  - icons/ folder with icon files

### 2. Load Extension in Chrome

1. Open Google Chrome browser

2. Go to Extensions page:
   - Type `chrome://extensions/` in the address bar
   - OR click Menu (â‹®) â†’ More Tools â†’ Extensions

3. Enable Developer Mode:
   - Look for "Developer mode" toggle in the top-right
   - Turn it ON

4. Load the extension:
   - Click "Load unpacked" button
   - Navigate to and select the `WP AUTOMATE` folder
   - Click "Select Folder"

5. Verify installation:
   - You should see "WhatsApp Message Automator" in your extensions list
   - The extension icon should appear in your Chrome toolbar
   - If you don't see it, click the puzzle icon and pin it

### 3. First Time Setup

1. Open WhatsApp Web:
   - Go to https://web.whatsapp.com
   - Scan QR code with your phone to log in

2. Test the extension:
   - Click the extension icon
   - The popup should open
   - Try entering a test number and message

### 4. Usage Tips

**Phone Number Format:**
```
+1234567890
+919876543210
+441234567890
```

**Recommended Settings:**
- Time Gap: 3-10 seconds
- Start with 1-2 test numbers
- Verify numbers before bulk sending

### 5. Troubleshooting

**Extension not appearing?**
- Refresh the extensions page
- Make sure all files are in the folder
- Check for any error messages

**Can't load extension?**
- Verify manifest.json is valid
- Check that Developer Mode is enabled
- Try restarting Chrome

**Not working on WhatsApp Web?**
- Make sure you're logged in
- Refresh the WhatsApp Web page
- Check browser console for errors (F12)
